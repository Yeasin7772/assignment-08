Discuss the scope of var, let, and const

var:

Var is the global variable is declare outside and a function. you can reassigned the value many time

let:

let variable is block scope. let declaration declares re-assignable optionally initializing each to a value.

const:

const variable is block scope. After setting its value you cannot reassigned the value a second time
